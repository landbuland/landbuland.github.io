  3. Check borded hessian metrix

    If the largest leading principal minors have the same sign as (−1)n, and the last n−m leading principal minor alternate in sign.

  4. Monotonic transformation of a concave function is quasi-concave.

  5. For utility function with 2 goods,

    U(x1,x2),U1,U2≥0

(b) Show that as ρ→0, this utility function comes to represent the same preferences as the (generalized) Cobb-Douglas utility function u(x)=x1α1x2α2. (c) Show that as ρ→−m, indifference curves become "right angles"; that is, this utility function has in the limit the indifference map of the Leontief utility function u(x1,x,)= Min{x1,x2}

    (19)u(x)=[α1x1ρ+α2x2ρ]1/ρ
