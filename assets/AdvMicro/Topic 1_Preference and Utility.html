1. Preference relations

  • Preference, At least as good as: ≿

  • Strict preference: ≻ : x≿y,y≿̸x.

  • Rational: The preference relation ≿ is rational if it possesses the following two properties:

    • Completeness: for all x,y∈X, we have that x≿y or y≿x (or both).

    • Transitivity: For all x,y,z∈X, if x≿y and y≿z, then x≿z.

  • If ≿ is rational, then

    • ≻ is both irreflexive ( x≻x never holds), and transitive (x≻y,y≻z⇒x≻z)

    • ∼ is reflexive, and transitive, and symmetric

    • if x≻y≿z, then x≿z.

 

Additional assumptions on preference.

  • Continuity : For every converging sequence of pairs of consumption bundles {(xn;yn)}n=1+∞ and limn→∞xn=x,limn→∞yn=y. If xn≿yn for ∀n, then x≿y.

    • A violation: Lexicographic preference（字典preference）

  • Local Non-satiation: For ∀x∈R+L, and ∀ϵ>0, ∃x such that d(x0,x)≤ϵ, x≻x0.

  • Monotone:

    • (MWG) The preference relation ≿ is monotone if x1≫x2 implies x1≻x2. It is strongly monotone if x1>x2 implies that x1≻x2.

    • (JR) The preference relation ≿ is strictly monotone if x1≥x2 implies x1≿x2 AND x1≫x2 implies x1≻x2. Votation

    • x1≥x2 if and only if ∀i,xi1≥xi2.

    • x1>x2 if and only if x1≥x2 and x1≠x2

    • x1≫x2 if and only if ∀i,xi1>xi2

  • Convexity: for ∀x∈X, the upper level set is convex

    • Strict convexity: x2≿x1 and x3≿x1, then tx2+(1−t)x3≻x1 for ∀t∈(0,1)

    • ⇒x2∼x1 and x3∼x1 then tx2+(1−t)x3≻x1 indifference

    • Indifference curves demonstrate diminishing marginal rate of substitution

    • preference is convex, but utility function is quasi-concave.

 

2. Utility function

Binary preference relation ⇒ utility function

Representation theorem:

  • Function U:X→R represents preference relation ≿ if for ∀(x1,x2)∈X×X,U(x1)≥U(x2)⇔x1≿x2

Convexity of Utility/Preference/Indifference curve:

The following statements are equivalent:

  • Utility function is quasi-concave

  • Preference is convex

  • Indifference curve is convex

  • The upper contour set is convex

 

3. Summary

For every rational (complete and transitive), continuous, strongly monotonic and (strictly) convex preference, we can find a continuous, strictly increasing and (strictly) quasi-concave utility function to represent it.

Utility functions are ordinal, and positive monotonic transform of a utility function can represent the same preference.
