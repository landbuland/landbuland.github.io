Continuity

Definition: A function f:Rn→R is continuous at a point x¯∈Rn if limx→x¯f(x)=f(x¯). Remark: f:Rn→R is continuous at x¯ if and only if

(1)∀ϵ>0:∃δ>0:∥x−x¯∥<δ⇒|f(x)−f(x¯)|<ϵ

Example

Example 1: Let λ∈R and define f:R→R by f(x)=λx, and let x¯∈R. To verify that f is continuous at x¯ : (i) If λ=0 we can simply let δ be any positive number, since |f(x)−f(x¯)|=0<ϵ for any ϵ. (ii) If λ≠0, then for any ϵ let δ=1|λ|ϵ; then

(2)|x−x¯|<δ⇔|x−x¯|<1|λ|ϵ⇔|λ||x−x¯|<ϵ⇔|λx−λx¯|<ϵ⇔|f(x)−f(x¯)|<ϵ

Theorem: Let x¯∈X. A function f:X→Y is continuous at x¯ if for every sequence {xn} that converges to x¯, the sequence {f(xn)} converges to f(x¯).

Proof: (⇐:) Let ϵ>0. Suppose, by way of contradiction, that there is no δ>0 such that dX(x,x¯)<δ⇒dY(f(x),f(x¯))<ϵ−i.e.,

(5)∀δ>0:∃x∈B(x¯,δ) for which f(x)∉B(f(x¯),ϵ)

Then, in particular, for every n∈N, let 1n play the role of δ above: there is an xn∈B(x¯,1n) for which f(xn)∉B(f(x¯),ϵ). We therefore have a sequence {xn} in X that converges to x¯ but the sequence {f(xn)} does not converge to f(x¯), contradicting our assumption that f is continuous.

(⇒:) Assume that (∗) holds, and let {xn} be a sequence that converges to x¯. In order to show that {f(xn)} converges to f(x¯), let ϵ>0. According to (∗), there is a δ>0 for which

(6)x∈B(x¯,δ)⇒f(x)∈B(f(x¯),ϵ)

Since {xn}→x¯, we can choose n¯∈N such that n>n¯⇒xn∈B(x¯,δ). But then

(7)n>n¯⇒f(xn)∈B(f(x¯),ϵ)

i.e., {f(xn)} converges to f(x¯), and f is therefore continuous at x¯.

Theorem: A function f:X→Y is continuous if and only if for every open set V in Y the inverse image f−1(V) is an open set in X.

Proof:

(⇒:) Suppose the function f is countinous, we want to show that f−1(V) is an open set in X.

Firstly, for any x¯, x¯∈f−1(V)⇒f(x¯)∈V.

Since V is open, ∀x∈V,∃ϵ>0:B(x,ϵ)⊆V. So B(f(x¯),ϵ)⊆V.

Next, since f is a countinuous function, ∀ϵ>0,∃δ>0:x∈B(x¯,δ)⇒f(x)∈B(f(x¯),ϵ), and thus f(x)∈V. Then x∈f−1(V).

So we derived x∈B(x¯,δ)⇒x∈f−1(V), which implies B(x¯,δ)⊆f−1(V). So f−1(V) is an open set.

(⇐:) Suppose every open set V in Y the inverse image f−1(V) is an open set in X. We want to show f is continuous.

For any point x¯ and any ϵ>0. Consider a small open ball B(f(x¯),ϵ), which is also a open set. Let V=B(f(x¯),ϵ)

Therefore, f−1(V) has to be a open set by our assumption, therefore, ∃δ>0:B(x¯,δ)⊆f−1(V).

And for any x∈B(x¯,δ)⇒x∈f−1(V)⇒f(x)∈V=B(f(x¯),ϵ)

Thus, we have ∀ϵ>0,∃δ>0:x∈B(x¯,δ)⇒f(x)∈B(f(x¯,ϵ)). f is countinous.

Propositions

  1. Sum and Scalar Multiple of Continuous Functions

Let f and g be continuous functions from a metric space (X,dX) to a normed vector space (Y,∥⋅∥), and let α∈R be a scalar.

  • Proposition (Sum): The function f+g, defined pointwise by (f+g)(x)=f(x)+g(x), is a continuous function from X to Y.

  • Proposition (Scalar Multiple): The function αf, defined pointwise by (αf)(x)=αf(x), is a continuous function from X to Y.

  1. Product and Reciprocal of Continuous Functions

These propositions apply specifically to real-valued functions, where the target space Y is R. Let e f and g be continuous functions from a metric space (X,dX) to the real numbers R.

  • Proposition (Product): The function fg, defined pointwise by (fg)(x)=f(x)g(x), is a continuous function from X to R. es

  • Proposition (Reciprocal): If f(x)≠0 for all x∈X, the function 1/f, defined pointwise by (1/f)(x)=1/f(x), is a continuous function from X to R.

  1. Composition of Continuous Functions

Let f:X→Y and g:Y→Z be continuous functions between metric spaces (X,dX), (Y,dY), and (Z,dZ).

  • Proposition (Composition): The composite function g∘f, defined by (g∘f)(x)= g(f(x)), is a continuous function from X to Z.

The Weierstrass theorem

In Euclidean space (i.e., Rn with any norm) we say that a set is compact if it's both closed and bounded. One of the most important properties of continuous functions is that they "preserve" compactness - i.e., if X is a compact subset of Rn and if f:X→Rm is a continuous function, then the image of X,f(X), is a compact set in Rm. This is the Weierstrass Theorem. In fact, the Weierstrass Theorem holds in general metric spaces:

Weierstrass Theorem: If X is compact and f:X→Y is continuous, then f(X) is a compact subset of Y.

Corollary: If f:X→R is a continuous real-valued function on a compact set, then f attains a maximum and a minimum on X.

Proof

As f(x) is continuous and x is compact. then x is closed and bounded. ⇒f(x) is closed & bounded. As f(x) is bounded, it must has a least upper bound M and a largest lower bound m.

As f(x) is closed. it has to contain all its limit points

So M,m∈f(x),⇒ M=max{f(x)} and m=min{f(x)}.

Consider the following utility maximization problem.

As u:R2→R is continuous and The constraint set is closed and bounded (it is compact). By the Weierstrass theorem we know there must exist a maximum and a minimum.
