Definition: A real-valued function f:X→R on a convex set X is concave if

(1)∀x,y∈X,λ∈(0,1):f((1−λ)x+λy)≧(1−λ)f(x)+λf(y}

and f is convex if

(2)∀x,y∈X,λ∈(0,1):f((1−λ)x+λy)≦(1−λ)f(x)+λf(y}.

 

Definition: A real-valued function f:X→R on a convex set X is quasiconcave if

(3)∀x,y∈X,λ∈(0,1):f((1−λ)x+λy)≧min{f(x),f(y)}

and f is quasiconvex if and only if

(4)∀x,y∈X,λ∈(0,1):f((1−λ)x+λy)≦max{f∣(x),f(y)}

Tip

It is important to understand the hierarchy and relationships between these different types of functions. Here are three key points to remember:

  • Quasiconcavity is a weaker condition than concavity. This means that while every concave function is also quasiconcave, the reverse is not true; there are functions that are quasiconcave but not concave.

  • It is also possible for a function to be neither concave nor convex. A classic example of this is the function f(x1,x2)=x1x2.

  • A special case occurs for functions defined on the real line, R. Any monotonic function (either increasing or decreasing) is both quasiconcave and quasiconvex.

 

Definition: Let f:X→R be a real-valued function on a set X⊆Rn, and let a∈R. The f-upper-contour set and the f-lower-contour set for a are the sets

(5)Uf(a)={x∈X∣f(x)≧a} and Lf(a)={x∈X∣f(x)≦a}

Let x∈X; in a slight abuse of notation, we say the f-upper-contour set and the f-lowercontour set of x are the sets Uf(f(x)) and Lf(f(x))-i.e.,

(6)Uf(x)={z∈X∣f(z)≧f(x)} and Lf(x)={z∈X∣f(z)≦f(x)}

The strict upper- and lower-contour sets are defined by replacing the weak inequalities with strict inequalities, and are denoted by Uf∘(⋅) and Lf∘(⋅). When we need to make the distinction, the sets defined by weak inequalities are called the weak upper- and lower contour sets.

Theorem: * A function f:X→R is quasiconcave if and only if all of its upper-contour sets are convex - i.e., if and only if ∀x∈X:Uf(x) is convex. And f is quasi convex if and only if all of its lower-contour sets are convex, ∀x∈X:Lf(x) is convex.

Proof

We prove when the function is quasi-concave.

⇒: f is quasiconcave, we want to show its uppercontour set is convex, i.e. we wanted to show ∀x,y∈Uf(a)⇒λx+(1−λ)y∈Uf(a).

Since f is quasiconcave, f(λx+(1−λ)y)≥min{f(x),f(y)}

By definition of upper contour set, x,y∈Uf(a)⇒f(x)≥a,f(y)≥a. So f(x+(1−λ)y)≥min{f(x),f(y)}≥a. This implies that λx+(1−λ)y∈Uf(a)

⇐: We want to show ∀x,y∈X,λ∈(0,1):f((1−λ)x+λy)≧min{f(x),f(y)}.

Let a=min{f(x),f(y)}, therefore, f(x)≥a,f(y)≥a⇒x,y∈Uf(a).

As every upper contour set is convex, λx+(1−λ)y∈Uf(a)⇒f(λx+(1−λ)y)≥a=min{f(x),f(y)}

 

Example: Let f:R++2→R be the function defined by f(x)=x1x2. Verify that f is neither concave nor convex, but is quasiconcave.

To prove that f is not concave, we need to find a single counterexample that violates the definition of a concave function. Let x=(1,1) and y=(3,3).

Now, let's check the values with λ=0.5 (the midpoint): z=0.5⋅(1,1)+0.5⋅(3,3)=(0.5,0.5)+(1.5,1.5)=(2,2). and 0.5⋅f(x)+0.5⋅f(y)=0.5⋅f(1,1)+0.5⋅f(3,3)=0.5⋅(1)+0.5⋅(9)=0.5+4.5=5.

We found that f(z)=4, which is less than the average value of 5 . This violates the inequality required for a concave function. Therefore, f is not concave.

Similarly, to prove that f is not convex, we need a counterexample that violates its definition.

Let x=(1,4) and y=(4,1)

z=0.5⋅(1,4)+0.5⋅(4,1)=(0.5,2)+(2,0.5)=(2.5,2.5), 0.5⋅f(x)+0.5⋅f(y)=0.5⋅f(1,4)+0.5⋅f(4,1)=0.5⋅(4)+0.5⋅(4)=2+2=4.

We found that f(z)=6.25, which is greater than the average value of 4 . This violates the inequality required for a convex function. Therefore, f is not convex.

To prove that f is strictly quasiconcave, the easiest method is to use the theorem that connects quasiconcavity to the convexity of its upper-contour sets.

  • Recall The Definition: A function is quasiconcave if all of its upper-contour sets are convex. It is strictly quasiconcave if for any two distinct points x and y such that f(x)=f(y), we have f((1−λ)x+λy)>f(x) for all λ∈(0,1).

An upper-contour set is defined as Uf(a)={x∈R++2∣f(x)≥a}, which for our function is the set of points where x1x2≥a (for some positive constant a).

This set represents the area on or above a rectangular hyperbola in the first quadrant. This is a classic example of a convex set. (In economics, this corresponds to the "at least as good as" set for a standard Cobb-Douglas utility function, which is always assumed to be convex).

Since all its upper-contour sets are convex, the function is quasiconcave.
