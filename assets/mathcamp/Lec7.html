Unconstraint optimization

Definition: Let f:X→R be a real-valued function and let x¯∈X. [Note that X can be any set, not necessarily a subset of Rn.] We say that x¯ is a (global) maximum or maximum point of f if ∀x∈X:f(x¯)≧f(x); and x¯ is a (global) minimum or minimum point of f if ∀x∈X:f(x¯)≦f(x). We say the x¯ is an extremum if it's either a maximum or a minimum.

Definition: Let f:X→R be a real-valued function on a set X in Rn and let x¯∈X. We say that x¯ is a local maximum of f if there is neighborhood N of x¯ such that ∀x∈N:f(x¯)≧f(x); x¯ is a local minimum of f if there is neighborhood N of x¯ such that ∀x∈N:f(x¯)≦f(x). We say that x¯ is a locally unique or strict maximum or minimum if there is a neighborhood N in which the inequality is strict for all x∈N except x¯ itself.

Obviously, a global maximum/minimum is a local maximum/minimum, and a local maximum/minimum is not necessarily a global maximum/minimum.

Theorem: Let f:R→R. If x¯ is a local maximum of f and f is differentiable at x¯, then x¯ is a critical point of f.

Proof: We apply the same argument as in the preceding proof to each of the partial derivatives of f. Suppose ∂f∂xi(x―)>0. Then there is a δ>0 such that

(2)0<|h|<δ⇒f(x―+hei)−f(x―)h>0

where ei is the i-th unit vector in Rn. Then for 0<h<δ we have f(x―+hei)−f(x―)>0− i.e., f(x―+hei)>f(x―). Therefore there is no neighborhood of x― on which f(x)≦f(x―), so x― is not a local maximum of f. Since this contradicts the assumption of the theorem, we cannot have ∂f∂xi(x―)>0. A parallel argument shows that we cannot have ∂f∂xi(x―)<0. Therefore ∂f∂xi(x―)=0.

Theorem: Let f:R→R and let x¯ be a point at which f is twice differentiable. If f′(x¯)=0 and f′′(x¯)>0, then x¯ is a local minimum point of f.

Proof: As we've done before, we define the function F by F(Δx)=f(x¯+Δx)−f(x¯). We'll show that for some neighborhood N around zero

(3)[Δx∈N&Δx≠0]⇒F(Δx)>0

i.e., f(x¯+Δx)>f(x¯) for all nonzero Δx∈N, so this will show that x¯ is indeed a locally unique minimum point of f.

Using the 2nd -degree Taylor polynomial, we have

(4)F(Δx)=f′(x¯)Δx+12f′′(x¯)(Δx)2+R2(Δx)

where R2(Δx) is the remainder term, which satisfies limΔx→01(Δx)2R2(Δx)=0. Because the theorem assumes that f′(x¯)=0, equation (1) becomes

(5)F(Δx)=12f′′(x¯)(Δx)2+R2(Δx)

and because we're excluding Δx=0, we can divide both sides of equation (2) by (Δx)2, so we have

(6)1(Δx)2F(Δx)=12f′′(x¯)+1(Δx)2R2(Δx)

Equation (3) is the key element of the proof: the two terms on the right-hand side are 12f′′(x¯), which is a given positive number, and 1(Δx)2R2(Δx), which is very small when Δx is small. So for small values of Δx, the first term dominates the second term - i.e., since the first term is a positive number, when we add the second term to it (even if the second term is negative), the right-hand side will still be positive. (Note, in fact, that when R2(Δx) is positive, the right-hand side is clearly positive, so it's only those Δx for which R2(Δx) is negative that we have to worry about.) This will guarantee that F(Δx) on the left-hand side is positive, which is the conclusion we're trying to establish.

Now let's write down formally what we described informally in the preceding paragraph. Since limΔx→01(Δx)2R2(Δx)=0, there is a δ>0 such that

(7)0<|Δx|<δ⇒|1(Δx)2R2(Δx)|<12f′′(x¯)

Therefore, when 0<|Δx|<δ we have

(8)12f′′(x¯)+1(Δx)2R2(Δx)>0

and therefore F(Δx)>0, i.e., f(x¯+Δx)>f(x¯), for all Δx such that 0<|Δx|<δ, completing the proof.

Theorem: Let f:Rn→R and let x― be a point at which f is twice differentiable. If ∇f(x―)=0 and the Hessian matrix Hf(x―) is negative definite, then x― is a locally unique maximum point of f.
